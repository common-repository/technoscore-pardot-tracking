== Description ==

Technoscore Pardot Tracking is  best plugin for WordPress to add Pardot Tracking code to footer part of Selected/All webpages. 

= More details are as follows:- =
* It adds Pardot Tracking code to particular pages or all pages.

== Installation ==

1. Use WordPress' built-in installer
2. Download the ZIP file from WordPress website
3. Extract the technoscore-pardot-tracking directory to your computer
4. Upload the technoscore-pardot-tracking directory to the /wp-content/plugins/ directory
5. Activate the plugin in the Plugin dashboard
